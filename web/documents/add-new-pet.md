# คู่มือการเพิ่มสายพันธุ์สัตว์เลี้ยงใหม่ (Adding a New Pet Specie)

เอกสารนี้รวบรวมขั้นตอนการเพิ่มสายพันธุ์สัตว์เลี้ยงใหม่เข้ามาในโปรเจกต์ Tamagotchi โดยจะแบ่งขั้นตอนออกเป็น 2 ส่วนหลัก คือการเพิ่มการตั้งค่าในโค้ด (Configuration) และการเตรียมไฟล์รูปภาพรูป (Assets)

## 1. การตั้งค่าในไฟล์ โค้ด (`helpers/petAssets.js`)
ไปที่ไฟล์ `helpers/petAssets.js` และมองหา Object ที่ชื่อว่า `PET_CATALOG` จากนั้นให้เพิ่ม `property` ใหม่โดยใช้ชื่อสายพันธุ์ (pet_id) ที่คุณต้องการ

ตัวอย่างการเพิ่มสายพันธุ์ชื่อ `"dragon"` ที่มีเฉพาะช่วงอายุ `teen`:

```javascript
const PET_CATALOG = {
  // ... สายพันธุ์อื่นๆ ...
  
  dragon: createClassicStage(160, ["idle"])
};
```
### คำอธิบายการสร้าง Stage:
ระบบมีฟังก์ชัน `createClassicStage(ขนาดภาพ, รายการท่าทาง, { options })` สำหรับช่วยตั้งค่า
- **พารามิเตอร์ 1 (ขนาด):** กำหนดขนาดในการแสดงผลบนหน้าจอ เช่น 132, 148, 160
- **พารามิเตอร์ 2 (ท่าทาง):** เป็น Array รายชื่อท่าทาง (Variants) ที่สายพันธุ์นี้มี เช่น `["idle"]` หรือถ้ามีครบก็ดึงตัวแปร `VARIANT_ORDER` ยัดเข้าไปเลย (`["idle", "attack", "sick", "angry", "dead"]`)
- **พารามิเตอร์ 3 (ออปชันเสริม):** สามารถละไว้ได้เลยสำหรับการตั้งค่าสายพันธุ์ใหม่

---

## 2. การเตรียมไฟล์รูปภาพภาพ (Assets)

หลังจากประกาศสายพันธุ์ในตัวโค้ดแล้ว จะต้องนำไฟล์ภาพ (Sprite SVG) ของสายพันธุ์นั้นไปจัดเก็บให้ถูกโฟลเดอร์ 

1. เข้าไปที่ไดเรกทอรี `assets/pet/`
2. สร้างโฟลเดอร์สำหรับ **สายพันธุ์** โดยตั้งให้ตรงกับคีย์ที่ประกาศใน `PET_CATALOG` (เช่น `dragon`)
3. นำไฟล์ภาพ .svg ไปวางไว้ข้างในโฟลเดอร์สายพันธุ์นั้นได้เลย โดยต้องตั้งชื่อไฟล์ให้ตรงกับท่าทาง ตัวอย่างเช่น:
   - `idle.svg`
   - `attack.svg`
   - `sick.svg`

**โครงสร้างโฟลเดอร์ตัวอย่าง:**
```text
assets/
└── pet/
    └── dragon/
        ├── idle.svg 
        ├── sick.svg 
        └── dead.svg
```

### คำแนะนำเกี่ยวกับการวาด SVG สำหรับตัวเกม
เราแนะนำให้วาดโดยยึดสไตล์ของ Sprite แบบดั้งเดิม (Classic Pattern) ซึ่งมีคุณสมบัติดังนี้:
- ขนาดแคนวาสและ viewBox เริ่มต้นที่ `128x128`
- เลือกระบุแอตทริบิวต์ `shape-rendering="crispEdges"` เพื่อให้ขอบของภาพมีความคมชัดแบบ Pixel Art
- สีโทนเข้ม (สีตัว): `#44514b`
- สีโทนอ่อน (สีตา หรือ ไฮไลต์): `#b7c7b5`
- ใช้แท็ก `<rect>` มารวมกันเพื่อสร้างเป็น pixel ขนาด 8x8 หรือทวีคูณของ 8 (เช่น `width="8" height="8"` หรือ `width="16" height="8"`)
